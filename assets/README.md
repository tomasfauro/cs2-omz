# Assets

Place the following files here:

- `logo.ico` — application icon used by `build.bat` / PyInstaller.
- `screenshots/main.png`, `screenshots/network.png`, `screenshots/launch.png` —
  referenced by the README.

The `logo.ico` file is intentionally left out of the repository so each fork
can supply its own. Generate one from a 512x512 PNG with any online converter
(e.g. `convertio.co/png-ico/`) and drop it here before running `build.bat`.
